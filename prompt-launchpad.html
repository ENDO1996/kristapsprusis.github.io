Prompt Launchpad
Portfolio

One prompt. Many tools.

This page helps you launch your prompt across multiple sites you are already logged into. Some sites allow pre-filled prompts, others require paste + submit due browser security rules.

Destinations

Tip: click a row to edit URL templates. Use {prompt} where the encoded prompt should be inserted.

5-minute setup (no coding needed)

  1. Download these files to one folder: prompt-launchpad.html, prompt-launchpad.js, styles.css.
  2. Windows: double-click run-launchpad.bat.
    Mac/Linux: run run-launchpad.command.
  3. Bookmark http://localhost:4173/prompt-launchpad.html.
  4. Allow popups for this page once, so multi-tab open works.
  5. Optional: in Chrome/Edge use “Install app” / “Create shortcut” for one-click opening.

How this works

  • For each selected site, this tool opens a new tab.
  • If a URL template contains {prompt}, the prompt is pre-filled in the URL.
  • If not, the site still opens and your prompt is copied so you can paste and hit generate.
  • Your destination list is saved in your browser (localStorage).
