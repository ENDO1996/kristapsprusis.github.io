# Prompt Launchpad (Beginner Quick Start)

This is a static web app. No build step needed.

## Files you need
- `prompt-launchpad.html`
- `prompt-launchpad.js`
- `styles.css`
- `run-launchpad.bat` (Windows quick launch)
- `run-launchpad.command` (Mac/Linux quick launch)

## Fastest way to run
### Windows
1. Keep all files in one folder.
2. Double-click `run-launchpad.bat`.
3. Your browser opens automatically.

### Mac/Linux
1. Keep all files in one folder.
2. Make script executable once: `chmod +x run-launchpad.command`
3. Run `./run-launchpad.command`

## If you downloaded from GitHub
- Click **Code** → **Download ZIP**.
- Extract the ZIP.
- Open the extracted folder and run one of the launcher scripts above.

## If you are using Codex
The files in this repo are already present in the workspace folder. If your GitHub is connected, commits can be pushed and then downloaded from GitHub too.
